## 使用docker部署gitlab
### 1、创建文件夹
```shell script
cd /usr/local/
mkdir gitlab
```
### 2、创建docker-compose.yml文件
```shell script
]# pwd
/usr/local/gitlab

]# cat docker-compose.yml

  redis:
    restart: always
    image: redis:5.0.9
    command:
    - --loglevel warning
    volumes:
    - /usr/local/gitlab/redis-data:/var/lib/redis:Z

  postgresql:
    restart: always
    image: sameersbn/postgresql:11-20200524
    volumes:
    - /usr/local/gitlab/postgresql-data:/var/lib/postgresql:Z
```

### 3、docker-compose里面的注意事项
#### 3.1 内外端口映射关系
宿主机暴露端口：容器内端口
```shell script
ports:
    - "13800:80"  ###http端口映射
    - "13822:22"  ###ssh端口映射
......
    - GITLAB_PORT=13800
    - GITLAB_SSH_PORT=13822
```
#### 3.2 暴露http连接的url
```shell script
 - GITLAB_HOST=172.16.120.120
```
#### 3.3 设置root初始密码和root的email
```shell script
    - GITLAB_ROOT_PASSWORD=XXXXXX
    - GITLAB_ROOT_EMAIL=498946975@qq.com
```
#### 3.4 挂载点
宿主机挂载点：容器内目录
```shell script
...
  redis:
    volumes:
    - /usr/local/gitlab/redis-data:/var/lib/redis:Z
...
  postgresql:
    volumes:
    - /usr/local/gitlab/postgresql-data:/var/lib/postgresql:Z
...
  gitlab:
    volumes:
    - /usr/local/gitlab/gitlab-data:/home/git/data:Z
...
```
### 4、docker-compose启动
-d:后台运行程序
```shell script
docker-compose up -d
```
